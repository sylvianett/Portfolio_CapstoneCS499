Artifact: CS 320 Appointment Service

This artifact is a Java service layer project created in CS 320 that supports creating, updating, and deleting appointment objects while enforcing required validation rules. The project includes unit tests written with JUnit to verify the service logic and validate that invalid input is handled correctly.

Created: CS 320 (Software Test, Automation & Quality Assurance)
